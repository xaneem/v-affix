# v-affix

## Install

``` bash
npm install v-affix --save
```

## Usage

CommonJS

```` javascript
var Affix = require('v-affix');

new Vue({
  components: {
    Affix: Affix
  }
});
````

ES6

```` javascript
import Affix from 'v-affix';

new Vue({
  components: {
    Affix
  }
});
````
